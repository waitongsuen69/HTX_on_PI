How to get a TRON API key (TronGrid)
  1. Go to https://www.trongrid.io/ and sign in or create an account.
  2. Open the Dashboard and create a new API key.
  3. Copy the value shown as TRON-PRO-API-KEY.
  4. Paste it into the “TRON API Key (required, shared)” field.
  5. Optional: keep the default Full Node or add your own endpoint.

This key is shared by all TRON DEX addresses in this app.

How to get a Blockfrost Project ID
  1. Go to https://blockfrost.io/ and sign in or create an account.
  2. Create a new project for Cardano mainnet.
  3. Copy the displayed Project ID (not the secret).
  4. Paste it into the “Blockfrost Project ID” field.

This ID is shared by all Cardano DEX accounts in this app.
